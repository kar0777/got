Что умеет Switcher

Не подменяет GitLab и не выдаёт подписку — он аккуратно связывает ваши профили, проекты и локальный рабочий контекст.

What Switcher does

It does not replace GitLab or grant a subscription. It connects your profiles, projects, and local working context in one practical workflow.

👥

Изолированные аккаунты

Isolated accounts

Каждый профиль получает отдельную OAuth-конфигурацию glab и собственную папку авторизации.

Every profile receives a separate glab OAuth configuration and authentication directory.

🧠

Локальная память

Local memory

Компактный transcript очищает терминальный шум и переносит полезный контекст к следующему профилю.

A compact transcript removes terminal noise and carries useful project context to the next profile.

📁

Несколько проектов

Multiple projects

Сохраняйте локальные Git-репозитории и запускайте Duo CLI сразу в правильной рабочей папке.

Save local Git repositories and launch Duo CLI directly in the correct working directory.

📊

Usage в одном окне

Usage in one view

Hub показывает доступные данные usage по аккаунтам, когда GitLab API их возвращает.

The Hub displays available account usage data whenever the GitLab API provides it.

🩺

Doctor и восстановление

Doctor and recovery

Проверка glab, OAuth, ConPTY-рекордера, контекста, памяти и прав локальных профилей.

Checks glab, OAuth, the ConPTY recorder, project context, memory, and local profile permissions.

🔐

Безопасные настройки

Safer defaults

Подтверждение команд включено, а потенциально чувствительные сырые VT-логи выключены.

Command confirmation is enabled, while potentially sensitive raw VT logs are disabled.
