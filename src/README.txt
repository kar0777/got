GITLAB DUO CLI SWITCHER 8.3.2 MINIMAL

В архиве только 4 файла:
- Run.cmd
- GitLabDuoCLI-Switcher.ps1
- DuoTerminalRecorder.cs
- README.txt

ИСПРАВЛЕНИЕ 8.3.2

Исправлены ошибки совместимости Windows PowerShell 5.1 и устойчивости конфигурации:
- оператор -and больше не начинается с новой строки;
- Test-Path больше не используется как голая команда внутри if;
- пустые объекты hooks.json больше не вызывают падение в StrictMode;
- повреждённые и неполные hooks.json автоматически восстанавливаются с backup;
- пустые/старые profiles.json корректно мигрируют;
- некорректный путь проекта не завершает Hub.

Run.cmd теперь перед запуском использует встроенный PowerShell parser.
При синтаксической ошибке Switcher не запускается и показывает номер строки.

ОБНОВЛЕНИЕ

1. Закройте Hub и GitLab Duo CLI.
2. Полностью распакуйте ZIP поверх папки программы.
3. Замените все четыре файла.
4. Запустите Run.cmd.

Аккаунты, OAuth, проекты и история сохраняются в:

%LOCALAPPDATA%\GitLabDuoCLISwitcher
